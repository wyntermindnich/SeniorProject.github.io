Binary:

Binary is a numeric system represented in zeros and ones, it is nothing more than a system of counting. It can also be called base-2, there are only two integer representations. Binary can represent any decimal. Binary can represent boolean logic, where there are only two representations, True (1) and False (0). Another way to think of boolean logic is like a switch, On (1) and Off (0). The ASCII stands for the American Standard Code for Information Interchange. It is a table for converting computer data in binary to letters, character, and symbols.

Functions

A function contains a sequence of commands that can later be used in code. It works by piecing together basic actions to carry out a greater task. The importance of this is that it allows repetition of code without having to rewrite all of the code.

Data Structures

    Data structures are a way to logically store and organize data for use. These structures can be filled with data types described above. A data structure can be selected or designed.Below are the most common data structures you will encounter.

  • Linear data structure - Organized sequentially

    • These are some common linear data stuctures programers use.

    • Array

      • Contains a collection of data type of similar type.

    • Stack

      • Ordered via LIFO, Last In First Out

    • Queue

      • Ordered via FIFO, First In First Out

    • Linked-list

      • A collection of nodes where each node in the list refers to the next node in the sequence.

  • Non-linear data structure - Organized non-sugentially, cannon be traversed in a single line/run

    • These are thw two most common non-linear data stuctures programers use.

    • Tree

      • A collection of linked nodes, with a parent child relationship.

    • Graph

      • A collection of vertices (points) and edges (lines). The edges connect the vertices, and represent the relationship of vertices.
